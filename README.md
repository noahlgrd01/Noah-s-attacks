# But de l'attaque #
Le but de l’attaque est de pouvoir siphonner des fichiers spécifiques provenant d’un système GNU/Linux, à l’aide d’un Rubber Ducky (RD).

Les fichiers et répertoires cibles sont

- cd ~/.ssh
- cd ~/.gnupg
- nano ~/.bash_history
- nano /etc/resolv.conf
- nano /etc/crontab
